# API

## Technologies or Prerequisite
- Mongo (Database used)

## To install and run project locally

    $ git https://github.com/lytes20/deselect-api.git
    $ cd deselect-api
    $ npm install

### In Dev or Local
    $ npm run dev

## Populate DB
    $ npm run populate


## Endpoints

`GET "/get/students" `
  
`GET "/get/students/by/nationality"`

`GET "/get/students/by/nationality/sort"`

`POST "/post/students"`
